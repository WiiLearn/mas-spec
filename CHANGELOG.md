# Changelog

All notable changes to the MAS Specification are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [3.0.0] - 2025-12-11

### Added
- **Part 11: Spec-Driven Artifacts** - Kiro-compatible spec-to-diagram generation
- **Business-Technical Bridge Layer** - Connect requirements to implementation
- **Cross-Layer Relationships** - `serves`, `realizes`, `traces`, `validates`, `impacts`, `decomposes`
- **Traceability diagrams** - Requirements-to-code linking
- **C4 Integration** - Multi-level architecture zoom support
- **EARS notation** - Easy Approach to Requirements Syntax for AI agents

### Changed
- Conformance classes expanded to 7 levels
- Enhanced AI agent integration patterns

---

## [2.6.0] - 2025-11-15

### Added
- **Part 10: ASCII Wireframes** - Text-based UI mockups
- 15+ widget types for wireframe design
- Accessibility annotations for wireframes
- Cross-reference support between wireframes and flows

---

## [2.5.0] - 2025-10-01

### Added
- **Part 8: UX Diagrams** - User experience modeling
  - User Journey diagrams with satisfaction scores
  - User Flow diagrams with decision paths
  - Wireframe Flow diagrams
  - Interaction Pattern diagrams
  - Accessibility Flow diagrams
  - Design System diagrams
- **Part 9: Business Operations** - Business process modeling
  - Business Process Flow diagrams
  - Decision Tree diagrams
  - Swimlane diagrams
  - Value Stream Map diagrams
  - Organizational Chart diagrams
  - Business Rule diagrams
- WCAG 2.1 accessibility compliance
- BPMN 2.0 compatibility

### Changed
- Conformance classes expanded to 5 levels
- Enhanced LLM integration protocol

---

## [2.0.0] - 2025-06-01

### Added
- Formal EBNF grammar for all diagram types
- Security validation framework
- Conformance classes (Minimal, Standard, Full)
- LLM integration protocol
- 7 technical diagram types:
  - Graph/Flowchart
  - Sequence Diagram
  - Class Diagram
  - State Diagram
  - ER Diagram
  - Gantt Chart
  - Pie Chart

### Changed
- Complete specification rewrite with formal grammar
- Semantic validation rules

---

## [1.0.0] - 2025-03-01

### Added
- Initial MAP (Mermaid Artifact Protocol) specification
- Basic diagram generation guidelines
- Simple validation rules
